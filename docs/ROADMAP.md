# Roadmap

## Milestone 1 — Reliable tower companion

Status: **playable and under continuing playtest**

- [x] shared game engine;
- [x] Windows packaging configuration;
- [x] iPad-compatible PWA;
- [x] local saves and export/import;
- [x] deterministic testing seeds;
- [x] baseline automated tests;
- [x] first successful Windows user build and playtest;
- [ ] complete manual-based regression test scenario;
- [ ] verify all pending-action edge cases through multiplayer playtests;
- [ ] code-signed public Windows releases, if funding/permissions make that practical.

## Milestone 2 — Rules verification

Status: **in progress — first rules build delivered in v0.3.0**

- [x] searchable in-app paraphrased rules manual;
- [x] contextual Relevant Rule button;
- [x] expanded automated test suite;
- [x] corrected Citadel endgame requirement;
- [x] corrected counterclockwise frontier direction;
- [x] corrected one-warrior retreat restriction;
- [x] corrected Bazaar haggling to support lower/unchanged/closed outcomes;
- [ ] firmware/emulator research;
- [ ] probability tables with evidence grades;
- [ ] classic and research profiles;
- [ ] optional event-frequency diagnostics for tower testing;
- [ ] sample-game deterministic regression fixture.

## Milestone 3 — Deluxe presentation

Status: **first complete presentation build delivered in v0.4.0**

- [x] original four-kingdom interactive landmark atlas;
- [x] original animated tower windows;
- [x] original synthesized sound cues and optional ambient drone;
- [x] reduced-motion, high-contrast, and larger-text modes;
- [x] guided in-app tutorial;
- [x] Classic 1981 and Deluxe 2026 presentation modes;
- [ ] exact territory graph and automatic pawn routing;
- [ ] commissioned or community-contributed original landscape illustration;
- [ ] expanded event-specific animation library.

## Milestone 4 — iPad refinement

- tablet-first board layout;
- iPad save-file workflow;
- optional native Tauri iOS target after Apple signing requirements are addressed;
- local-network display mode for showing the Tower on a shared screen.


## Completed in v0.5.0

- Three-column desktop tabletop layout
- Event-specific Relevant Rule labels
